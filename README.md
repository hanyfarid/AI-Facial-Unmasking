# AI-Facial-Unmasking
AI-Powered Facial Unmasking

The data file biometric-scores.txt provides the biometric scores between pairs faces. The first numeric column is the condition (see below), the second column is the filename used for comparison, and the third column is the biometric in the range of -1 (mismatch) to 1 (perfect match).

Condiation:
1  same identity
2  different identity (same race/gender)
3  same identity (chatGPT unmasked)
4  same identity (chatGPT unmasked US Senators)
5  same identity (Grok unmasked)
6  same identity (Gemini unmasked)
7  same identity (Gemini unmasked US Senators)
8  different identity Doopelgangers
